## Descriptions
사내 기술역량강화교육으로 배운 React를 직원 정보 검색, 회의실 예약시스템을 만들었습니다.<br/>
Node.js 교육도 있었기에 서버 연동도 가능했지만, 완벽하게 숙지하지 못해 프론트엔드 코드로 재수정하였습니다.<br/>
디바이스 크기에 맞게 반응형 사이트로 제작했습니다.
